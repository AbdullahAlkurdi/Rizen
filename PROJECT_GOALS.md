Building Rizen, an AI-powered Life Operating System focused on discipline, routines, habit formation, burnout prevention, behavioral analytics, spiritual integration, and adaptive daily planning.

The platform includes:
- Dynamic routine generation
- Prayer-relative time blocking
- 7 life-domain trackers
- Positive and negative habit engine
- AI behavioral coach powered by Gemini
- Journaling and reflections
- Growth analytics
- Real-life reward system
- Developer CLI integration

Technology stack:
Flutter, Firebase, Clean Architecture, BLoC, Gemini API, Aladhan API.

Goal:
Generate complete production-ready product specifications, UX/UI flows, database architecture, system design, implementation plans, prompts, and development documentation for a world-class application.
