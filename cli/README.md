# RizenOS CLI — Part 3/4

This directory contains the terminal interface for RizenOS. It provides direct access to habits, todos, routines, and analytics from the command line.

## Setup

1. `cd cli`
2. `dart pub get`
3. Ensure `.env` exists in the project root with Firebase credentials.
4. Run `dart run rizen` or `dart run bin/rizen.dart`

## Commands

- `login` / `logout`
- `status`, `log`, `habit`, `todo`, `routine`, `analytics`
- `help`

Full command documentation will be completed in Part 4/4.
